+ 2주 마다 오프라인 만남
+ 각자 공부한 결과물을 `회차별 폴더`에 문서로 commit 하면 됩니다.
  + .md파일을 사용
  + 작성예시) 폴더명_이름.md
+ 결과물은 회차별 종료일자까지 올려야합니다.
